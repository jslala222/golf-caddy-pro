"서버비 0원, 관리 스트레스 Zero. 오직 캐디만을 위한 독립형 비서"

1. 프로젝트 개요

제품 형태: 클라이언트 사이드 웹 앱 (PWA) 또는 안드로이드 패키징 앱

타겟 사용자: 현직 골프 캐디 (프리랜서 특성상 수입/스케줄 관리가 필수인 층)

핵심 가치: 1.  철저한 보안: 내 수입 정보가 서버에 남지 않고 내 폰에만 저장됨.
2.  데이터 소유권: 폰을 바꿔도 내가 백업 파일을 가지고 있으면 언제든 복구 가능.
3.  직관적 사용: 바쁜 라운드 중에도 한 손으로 입력 가능.

2. 주요 기능 상세 (User Requirement 반영)

A. 📝 통합 스케줄러 (Schedule Table)

기능 정의: 근무 일정과 개인적인 약속을 하나의 달력에서 구분하여 관리.

데이터 구조:

날짜/시간: 필수 입력

구분: [⛳️ 근무] vs [📅 개인 약속] (토글 버튼)

내용: "1부 첫 팀", "치과 예약", "동창회" 등

알림: (추후 고도화 시) 푸시 알림 기능

B. 👥 고객 관리 데이터베이스 (Customer DB Table)

기능 정의: 라운드 전후 고객 성향을 기록하여 블랙컨슈머(진상) 대응 및 우수 고객 관리.

핵심 구분:

😡 진상(Blacklist): 빨간색 테두리 표시. (예: "야한 농담", "진행 재촉 심함", "욕설")

😇 천사(VIP): 파란색/골드 테두리 표시. (예: "팁 후함", "매너 좋음", "간식 챙겨줌")

검색 기능: 라운드 배정받자마자 고객 이름 검색 -> 진상 여부 미리 파악.

C. 💰 수입/지출 관리 (Accounting Table)

수입 관리:

캐디피(Official): 현금/이체 구분

오버피(Tips): 별도 입력 (이게 진짜 수익)

지출 관리 (요청하신 용품비 포함):

카테고리: [🧢 용품 구입], [🍱 식대], [🚗 유류비], [기타]

순수익 계산: (캐디피 + 오버피) - (용품비 + 식대 등) = "이번 달 진짜 번 돈"

D. 💾 데이터 백업 및 복구 (Business Core)

백업(Export): 사용자가 버튼을 누르면 JSON 파일 형태로 데이터 다운로드.

복구(Import): 백업해 둔 파일을 선택하면 앱에 데이터가 그대로 로드됨.

전략: 이 기능을 통해 기기 변경 시 유료 AS(데이터 이관 대행)를 유도.

3. 기술 스택 및 구조 (Technical Stack)

Frontend: React.js (단일 페이지 애플리케이션, SPA)

Database: Local Storage (브라우저 저장소) * 서버(DB)를 쓰지 않으므로 유지 보수 비용이 0원입니다.

Hosting: Netlify / Vercel (무료 티어 사용) 또는 안드로이드 APK 파일 직접 전송.

4. 수익화 모델 (Business Model)

💵 1단계: 프로그램 판매 (License Fee)

가격: 120,000원 (1회 구매로 평생 사용 강조)

판매 방식: 1.  입금 확인.
2.  개인별 고유 URL 또는 설치 파일(APK) 전송.
3.  간단한 사용 매뉴얼(PDF/이미지) 전송.

🛠 2단계: AS 및 유지 보수 (Retention)

상황: 핸드폰 분실, 기기 변경, 실수로 앱 삭제 시.

서비스: "데이터 이관/복구 서비스"

비용: 건당 20,000원 ~ 30,000원

논리: "고객님, 데이터는 고객님 폰에만 있어서 보안상 안전하지만, 옮기려면 전문가의 손길이 필요합니다." -> 백업 파일을 받아 새 기기에 세팅해 줌.
