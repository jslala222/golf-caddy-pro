# 🚀 캐디 매니저 프로 - 실시간 자동 배포 가이드

대표님, 이제 **"저장만 하면 자동으로 내 폰 앱이 바뀌는"** 마법 같은 세상을 만들어봅시다! 깃허브(GitHub)와 버셀(Vercel)을 연결하는 방법입니다.

---

## 1단계: 깃허브로 코드 보내기 (오늘만 한 번!)
1. 프로젝트 폴더에 있는 **`upload.bat`** 파일을 더블 클릭하세요.
2. 검은 창이 뜨고 글자가 올라갑니다.
3. 중간에 **웹 브라우저가 열리며 깃허브 로그인**을 물어보면, 로그인(또는 인증 버튼)을 눌러주세요.
4. "✅ 깃허브 업로드 완료!" 메시지가 뜨면 성공입니다!

## 2단계: 버셀(Vercel)과 창고 연결하기
1. **[Vercel 접속](https://vercel.com/)** 후 로그인하세요.
2. 메인 화면에서 **[Add New]** -> **[Project]** 클릭!
3. **[Import]** 버튼을 누르고 `jslala222/golf-caddy-pro` 창고를 선택하세요.
4. 맨 아래 **[Deploy]** 버튼을 누르면 끝! (약 1분 소요)

## 3단계: 이제 끝났습니다! 🎉
- 이제부터는 제가 코드를 고치고 저장만 하면, 버셀이 알아서 새 버전을 만듭니다.
- 약 1분 뒤에 폰에서 앱을 다시 열면 자동으로 최신 기능이 적용되어 있습니다!

---

## 🆘 카카오톡으로 링크를 보냈을 때 (중요!)
카카오톡 안에서는 보안 정책상 **"홈 화면 추가"** 버튼이 나타나지 않습니다. 와이프분이나 지인분께 보낼 때 이렇게 알려주세요:

1. 카톡에서 링크를 클릭해 창을 연다.
2. 우측 상단의 **[⋮] (점 3개)** 또는 **[공유 버튼]**을 누른다.
3. **"다른 브라우저로 열기"** 또는 **"Safari/Chrome으로 열기"**를 선택한다.
4. 이제 열린 브라우저에서 아래 4단계를 진행한다.

---

## 4단계: 폰에 설치하기 📱
1. 버셀이 준 주소로 폰에서 접속하세요.
2. (아이폰) 하단 공유 버튼 -> **"홈 화면에 추가"**
3. (갤럭시) 우측 상단 메뉴 -> **"홈 화면에 추가"** (또는 앱 설치)

이제 대표님의 앱은 깃허브라는 든든한 창고와 연결되어 언제 어디서든 안전하게 관리됩니다! ⛳️✨
