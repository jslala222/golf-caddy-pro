# 🚀 캐디 매니저 프로 - 실시간 자동 배포 가이드

대표님, 이제 **"저장만 하면 자동으로 내 폰 앱이 바뀌는"** 마법 같은 세상을 만들어봅시다! 깃허브(GitHub)와 네트리파이(Netlify)를 연결하는 방법입니다.

---

## 1단계: 깃허브로 코드 보내기 (오늘만 한 번!)
1. 프로젝트 폴더에 있는 **`upload.bat`** 파일을 더블 클릭하세요.
2. 검은 창이 뜨고 글자가 올라갑니다.
3. 중간에 **웹 브라우저가 열리며 깃허브 로그인**을 물어보면, 로그인(또는 인증 버튼)을 눌러주세요.
4. "✅ 깃허브 업로드 완료!" 메시지가 뜨면 성공입니다!

## 2단계: 네트리파이와 창고 연결하기
1. **[Netlify 접속](https://app.netlify.com/)** 후 로그인하세요.
2. 메인 화면에서 **[Add new site]** -> **[Import an existing project]** 클릭!
3. **[Deploy with GitHub]** 버튼을 누르고 `jslala222/golf-caddy-pro` 창고를 선택하세요.
4. **설정 확인 (자동으로 채워집니다)**:
   - Build command: `npm run build`
   - Publish directory: `out`
5. 맨 아래 **[Deploy golf-caddy-pro]** 버튼을 누르면 끝!

## 3단계: 이제 끝났습니다! 🎉
- 이제부터는 제가 코드를 고치고 저장만 하면, 네트리파이가 알아서 새 버전을 만듭니다.
- 약 1분 뒤에 폰에서 앱을 다시 열면 자동으로 최신 기능이 적용되어 있습니다!

---

## 4단계: 폰에 설치하기 📱
1. 네트리파이가 준 주소로 폰에서 접속하세요.
2. (아이폰) 하단 공유 버튼 -> **"홈 화면에 추가"**
3. (갤럭시) 우측 상단 메뉴 -> **"홈 화면에 추가"** (또는 앱 설치)

이제 대표님의 앱은 깃허브라는 든든한 창고와 연결되어 언제 어디서든 안전하게 관리됩니다! ⛳️✨
